# 理财监控中心

一个简洁高效的理财监控应用，帮助用户统一管理支付宝和银行的理财产品，实时查看收益情况，无需在多个APP之间来回切换。

## 功能特点

- **资产总览**：一目了然地查看总资产、总收益和今日收益
- **收益统计**：按周/月/年查看收益统计，对比支付宝和银行产品收益
- **产品列表**：统一展示所有理财产品，支持按平台筛选、搜索和排序
- **收益趋势**：通过图表直观展示日/周/月收益走势
- **收益提醒**：设置收益阈值，超过时自动提醒

## 技术实现

- 纯HTML5 + CSS3 + JavaScript实现
- 使用Tailwind CSS构建响应式界面
- 使用Chart.js绘制收益趋势图表
- 使用localStorage存储模拟数据

## 如何使用

1. 直接打开`index.html`文件即可运行应用
2. 应用会自动生成模拟数据，展示在界面上
3. 点击"刷新数据"按钮可以更新产品收益数据
4. 点击产品卡片可以查看产品详情
5. 在"收益提醒"部分可以开启提醒并设置阈值

## 数据说明

应用使用模拟数据展示功能，包括：

- 6个模拟理财产品（3个支付宝产品，3个银行产品）
- 30天的历史收益数据
- 每次刷新会根据产品收益率和随机波动更新收益数据
